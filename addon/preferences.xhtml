Important: All settings on this page require a restart of Zotero to take full effect. Modifier Keys Select at least one modifier key (Ctrl, Alt, Shift). Main Key Enter a single letter or number as the main key for the hotkey (e.g., 'K'). Auto-hide on Startup If enabled, Zotero will automatically hide its main window on startup, showing only the tray icon. This is mainly intended for "silent startup" when Zotero is launched on system login. Communication Port The local port used for communication between the plugin and the tray helper process. It is not recommended to change this unless you encounter a port conflict. Restart Zotero after changing.
